# 饮食推荐系统 - 前端

基于 Vue 3 + Element Plus 的个性化饮食推荐系统前端界面。

## 功能模块

- 用户注册/登录（支持小眼睛切换密码）
- 注册后饮食偏好选择弹窗
- 个人主页（信息展示与修改）
- 饮食推荐（调用大模型）
- 饮食计划（三餐展示、重新生成、日期切换）
- AI 助手（大模型对话）
- 饮食打卡（日历打卡）
- 营养统计（ECharts 热量趋势图）
- 偏好设置（批量编辑）

## 技术栈

- Vue 3
- Element Plus（UI 组件库）
- ECharts（图表）
- Axios（HTTP 请求）
- Vue Router（路由）

## 运行

```bash
npm install
npm run dev
